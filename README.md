# Paperboy

### Way to start

- Download and start a grpc-greeter server from the grpc-go [repo](https://github.com/grpc/grpc-go/tree/master/examples/helloworld)
- Go to the server code dir `cd ./grpc-go/examples/helloworld/greeter_server`
- Build and run it with `go build && cd ../ && ./greeter_server`
- Start the `paperboy` and check server addres in the client part of the `paperboy`
